# 🖼️ Image Tools — Free Online Image Utilities

Free online image tools for compressing, converting, resizing, cropping, and background removal. 100% client-side processing.

**GitHub**: https://github.com/lihualih/image-tools

---

## 🛠️ Tools Included

| Tool | Description |
|------|-------------|
| **Image Compressor** | Reduce image file size while maintaining quality |
| **Image Converter** | Convert between JPG, PNG, WebP, GIF, BMP |
| **Image Resizer** | Resize images to custom dimensions |
| **Image Cropper** | Crop images with visual selection |
| **Background Remover** | Remove solid color backgrounds |

## ✨ Features

- ✅ 100% client-side processing (no file upload)
- ✅ Drag and drop support
- ✅ Before/after preview
- ✅ Batch processing support
- ✅ Mobile responsive
- ✅ Dark mode
- ✅ Download processed images

## ⚡ Tech Stack

- HTML5 / CSS3 / Vanilla JavaScript
- **Canvas API** — Image manipulation (compress, convert, resize, crop)
- Zero external dependencies

## 🚀 How to Use

1. Select the tool you need
2. Upload your image(s) (drag & drop or click)
3. Adjust settings (quality, dimensions, format)
4. Preview the result
5. Download
